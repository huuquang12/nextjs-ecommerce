# Getting started
- Clone the repository
```
git clone https://github.com/huuquang12/nextjs-ecommerce.git
```
- Install dependencies
```

- Backend
cd backend
npm install
```
- Build and run backend
```
npm start
```
Navigate to `http://localhost:8000`

- Frontend
cd frontend
npm install
```
- Build and run backend
```
npm run dev
```
Navigate to `http://localhost:3000`
